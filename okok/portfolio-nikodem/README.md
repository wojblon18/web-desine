# Portfolio of Nikodem Świeton

## Overview
This portfolio showcases the work and skills of Nikodem Świeton, a passionate individual in the field of web development. The website includes various sections that highlight personal information, technical skills, projects, and a contact form.

## Project Structure
The project consists of the following files:

- **index.html**: The main entry point for the portfolio, featuring navigation links to "O mnie," "Umiejętności," "Kontakt," and "Projekty." The title is set to "Nikodem Świeton."
  
- **o-mnie.html**: Contains information about Nikodem Świeton, including a profile picture and a brief introduction.

- **umiejetnosci.html**: Displays the title "Moje umiejętności," a brief introduction, and a section showcasing technical skills as badges. The layout utilizes CSS Grid for structure and Flexbox for the skills section. It also includes a brief description of the differences between Flexbox and CSS Grid.

- **projekty.html**: Showcases Nikodem Świeton's projects, featuring a filterable gallery of work.

- **kontakt.html**: Contains a contact form for visitors to reach out to Nikodem Świeton.

- **style.css**: Contains styles for the entire portfolio, including layout styles for CSS Grid and Flexbox, as well as styles for badges and other elements.

- **scripts/main.js**: Contains JavaScript for interactivity, including form validation, navigation, and any dynamic features needed for the portfolio.

## Setup Instructions
1. Clone the repository or download the project files.
2. Open the project folder in your preferred web browser.
3. Navigate through the portfolio using the provided links.

## Features
- Responsive design using CSS Grid and Flexbox.
- Interactive elements powered by JavaScript.
- Filterable project gallery.
- Contact form with validation.
